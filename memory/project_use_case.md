The timing-trainer.lua plugin was built specifically for the user to practice playing in time with a metronome. Output CC pans left when playing ahead of the beat, right when behind.

**Why:** User wants real-time visual/audio feedback on their rhythmic accuracy.
**How to apply:** Features that aid practice feedback (visual, audio, timing) are high-value requests for this project.
